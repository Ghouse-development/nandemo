# 🚀 クイックスタートガイド

設計・IC業務統合管理システムを5分でセットアップ！

## ステップ1: Supabaseテーブル作成（3分）

### 1-1. Supabaseダッシュボードを開く

ブラウザで以下のURLにアクセス：
```
https://twzsirpfudqwboeyakta.supabase.co
```

### 1-2. SQLエディタを開く

左側のメニューから「**SQL Editor**」をクリック

### 1-3. テーブル作成SQLを実行

1. 「**New query**」ボタンをクリック
2. `supabase_setup.sql` ファイルを開く
3. **全ての内容をコピー**
4. SQLエディタに**ペースト**
5. 「**Run**」ボタンをクリック
6. ✅ Success と表示されればOK

### 1-4. 初期データを投入

1. 再度「**New query**」ボタンをクリック
2. `supabase_seed_data.sql` ファイルを開く
3. **全ての内容をコピー**
4. SQLエディタに**ペースト**
5. 「**Run**」ボタンをクリック
6. ✅ Success と表示されればOK

### 1-5. テーブル確認

左側のメニューから「**Table Editor**」をクリック

以下の5つのテーブルが作成されていることを確認：
- ✅ designers
- ✅ projects
- ✅ email_templates
- ✅ template_vendors
- ✅ task_template_mappings

## ステップ2: アプリケーション起動（1分）

### 2-1. HTMLファイルを開く

`index.html` をブラウザで開く

（ダブルクリック、または右クリック→「プログラムから開く」→ブラウザを選択）

### 2-2. 初期データ読み込み確認

画面が表示されたら、右上のステータスを確認：
- ✅ 「保存済み」と表示されればOK
- ❌ エラーが表示された場合は、ブラウザのコンソール（F12キー）を確認

## ステップ3: 初期設定（1分）

### 3-1. 設計士を追加

1. **📋 案件管理**タブを開く
2. 「**設計士管理**」ボタンをクリック
3. 設計士名を入力（例: 佐藤建築士）
4. 「**追加**」をクリック
5. 必要な人数分繰り返す
6. 「**完了**」をクリック

### 3-2. タスクとメールテンプレートを紐づける

1. **🔗 タスク設定**タブを開く
2. 各タスクにメールテンプレートを設定：

推奨設定：
| タスク | メールテンプレート |
|--------|-------------------|
| 換気 | パナソニックリビング近畿（換気図面） |
| 給排水 | 給排水設備業者（給排水設備図面） |
| サッシ依頼 | 小倉サンダイン（サッシプレゼン） |
| 太陽光 | 千博産業（evoltz配置図） |
| エボルツ | 千博産業（evoltz配置図） |

3. 「**設定を保存**」をクリック

## ✅ セットアップ完了！

これで準備は完了です。早速使ってみましょう！

## 🆘 困ったときは

### データが表示されない

**確認事項:**
1. インターネット接続は正常ですか？
2. Supabaseのテーブルは正しく作成されましたか？
   - Supabaseダッシュボード → Table Editor で確認
3. ブラウザのコンソール（F12キー）にエラーが表示されていませんか？

**解決方法:**
- ページを再読み込み（Ctrl + R または Cmd + R）
- ブラウザのキャッシュをクリア
- 別のブラウザで試す

### メール作成ボタン（📧）が表示されない

**原因:**
タスクとメールテンプレートの紐づけが設定されていません。

**解決方法:**
1. **🔗 タスク設定**タブを開く
2. 該当するタスクにメールテンプレートを設定
3. 「**設定を保存**」をクリック
4. **📋 案件管理**タブに戻る

### 保存が失敗する

**確認事項:**
1. インターネット接続は正常ですか？
2. 画面右上のステータスは何と表示されていますか？

**解決方法:**
- インターネット接続を確認
- ページを再読み込み
- 数秒待ってから再度保存を試す
